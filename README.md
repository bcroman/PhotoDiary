College Mobile Development Assessment

Using Android Studio to create an android application that allows the user to take a photo using the phone camera then allowing the user to add text to the photo. The photo file path and photo note will be saved into a local database which can be view a page on where all photos will be taken and the user can select a photo to view more details like the note and maybe location

App Functions:
- Have a Home Page
- Have a Camera Page
- Have a Gallery Page
- Have a Local Database
- Create a project folder for photo storage
- Have a nav bar for opening pages
- Create a App Logo
- Request For Permissions at Launch

Camera Functions:
- Switch Back and Front
- Flash Toggle
- Tap to Focus
- Pinch to Zoom
- Enable HDR Mode
- Save image as PNG
- Have a button to open Gallery Page

Database Functions:
- Save a local Copy
- Store Photo Path and photo Text
- Gallery Function
- View All Photos
- View a selected Photo
- Edit(Update) Text

Optional Features:
- Use and store photo location
- Create a Share Button to share image to other people/other apps
- Create tags for images
- Sort filters for the gallery page like A-Z or Z-A on title
